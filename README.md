This project performs exploratory data analysis (EDA) on crisis and disaster management data to uncover insights related to disaster severity, response effectiveness, economic impact, and recovery patterns. The analysis focuses on understanding how response time, aid allocation, and disaster types influence casualties and recovery outcomes.

Key aspects studied include response delay levels, recovery days distribution, economic loss trends, casualty patterns, and response efficiency.

---

## Features

- Data cleaning and preprocessing including handling missing values and date formatting  
- Feature engineering such as response delay categorization and recovery analysis  
- Statistical and conditional analysis of disaster impacts  
- Visualizations using histograms, box plots, violin plots, and bar charts  
- Insights into response effectiveness and disaster recovery behavior  

---

## Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn

1. Clone the repository  
2. Install the required dependencies
